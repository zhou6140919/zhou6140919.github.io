# Introduction

## What is crawler?

- By coding programs, simulating the process of a browser surfing the Internet, then let it fetch data.

## Risks

- May disturb the normal operation of the visited website
- May have captured specific types of information or data protected by law

**robots.txt shows which kinds of information that cannot be downloaded legally,**

## Taxonomy

- Normal Crawler: fetch the important section of the system, fetch the whole page data.
- Focus Crawler: fetch the disignated section of the page.
- Incremental Crawler: fetch the updated part of the website.

# Requests Package


'''python
import pandas as pd
import time
import requests
from bs4 import BeautifulSoup
from lxml import etree
from tqdm import tqdm

model_name = ["mengzi-bert-base", "mengzi-oscar-base", "mengzi-t5-base",
              "mengzi-bert-base-fin", "mengzi-oscar-base-retrieval", "mengzi-oscar-base-caption"]

url_dict = {
    model: f'https://huggingface.co/Langboat/{model}' for model in model_name}

download_num = {model: 0 for model in model_name}

for model in tqdm(url_dict, total=len(url_dict)):
    response = requests.get(url=url_dict[model])
    soup = BeautifulSoup(response.content, 'html.parser')
    selector = etree.HTML(str(soup))
    links = selector.xpath("/html/body/div/main/div/section[2]/div[2]/dl/dd")
    download_num[model] = links[0].text

print(download_num)

_time = time.strftime("%Y-%m-%d %H:%M:%S")
print(_time)


df = pd.DataFrame(download_num, index=[
                  _time], columns=list(download_num.keys()))

df.to_csv("download_num.csv", mode='a', header=False)
print("Wrote successful!")
'''
