## Installation

```bash
pip install icecream
```

## Usage

```python
from icecream import ic

text = 'abc'
ic(text)

def add(a, b):
    return a + b

ic(add(1, 2))
```

Output:

```bash
ic| text: 'abc'
ic| add(1, 2): 3
```
